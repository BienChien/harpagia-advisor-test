About
This tool uses Dynamic Programming to solve the optimal enchanting policy across all possible game states, guaranteeing the best possible recommendation at every step. It targets a distribution of 9 / 6 / 3 successes across the three affixes — assigned dynamically, so whichever affix is best positioned becomes the 9-target, and so on. If the current state makes that distribution unreachable, the result will show 9/6/3 Unachievable.
